# Debian-Velocity
Debian-Velocity: Intelligent Linux optimizer maximizing responsiveness on limited RAM. Implements a hybrid strategy: High-Priority ZRAM (Primary) + SSD Swap (Backup). Features TCP BBR, 'noatime' I/O, bloat removal, and Performance CPU mode. Includes menu-driven Auto-Config and Uninstaller. Built to keep pentesting rigs stable under heavy load.
